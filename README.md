# react-skipio

A simple app showing a list of contacts and allows you to send them an SMS message.

## Setup

Create a `src/credentials.json` file with the following and replace `123abc` with your skipio token.

```json
{
  "token": "123abc",
  "environment": "stage"
}
```

## Running

```
npm start
```
